When you blunder a **rank 1 or higher** arcane spell, roll 1d100 and apply the result immediately. All effects are temporary.

**Targeting:** Choose nearest valid target by distance; if tied, random. Default to caster if no valid target exists.

**Scaling:** Effects list values for spell ranks 1/2/3/4/5 where applicable.

**Random Locations:** When an effect teleports creatures to "random locations," the GM chooses valid unoccupied spaces within the specified range, distributing them evenly around the area or rolling randomly for each creature's direction and distance.

## How to adjudicate

- The player rolls the d100, but the GM reads the result and describes how it manifests. The roll determines the effect; the GM decides the concrete placement, timing, and any narrative detail necessary to resolve it fairly.
- Targeting: by default use the table's targeting guidance (nearest valid target by distance; if tied, random). If the effect covers an area, the GM chooses the centre point (or roll randomly). The GM may use one of these deterministic or random methods when multiple valid targets exist:
	- Nearest: choose the nearest valid target by distance from the caster. If multiple are tied, roll 1dX among tied targets.
	- Initiative index: select the Nth creature in initiative order (roll 1dN). This tends to bias toward currently active combatants.
	- Random distribution: roll 1d8 for direction and 1d4 for distance to place affected points (useful for scatter/teleport effects).
	- Spread evenly: when the effect teleports or scatters multiple creatures, distribute them evenly around a chosen point or roll randomly per creature if the table says so.
- Tie-breakers: always break ties randomly (roll among tied candidates) unless the GM decides otherwise for narrative reasons.
- Cascading: some entries intentionally spawn additional table rolls (for example, Wild Zone and Reality Ripple). Those specific entries explicitly say they "don't cascade" â€” treat them as single additional rolls only. Never allow infinite recursion; if multiple roll-triggers would cascade, resolve only one extra roll unless the table explicitly permits otherwise.
- GM improvisation: if a result calls for a "random location" or ambiguous surreal consequence, favor descriptive weirdness over strict mechanics; use short adjudication rolls (1d4/1d6) to decide direction, distance, or which micro-anomaly occurs.

## Optional: Permanent Mutations

**GM Option:** For campaigns where wild surge exposure leaves lasting marks, each time a caster triggers a wild surge effect, they may develop a permanent mutation. Roll 1d12 on the Mutation Table below and advance that mutation by one stage. If you roll a mutation you've already progressed, advance it to the next stage (up to Stage 4 maximum).

### Mutation Stages

Each mutation progresses through four stages with repeated exposure:

- **Stage 1 (Subtle):** Barely noticeable cosmetic change
- **Stage 2 (Noticeable):** Obvious physical alteration
- **Stage 3 (Empowered):** Cosmetic intensifies + minor trade-off (small flaw and trait)
- **Stage 4 (Transformed):** Full trade-off (significant flaw and powerful trait)

### GM Guidance

- **Frequency:** Each wild surge trigger prompts a mutation roll. For campaigns where this feels too fast, consider requiring multiple triggers before rolling (e.g., every 3rd or 5th wild surge effect).
- **Player Choice:** Consider letting players choose their mutation (with GM approval) instead of rolling randomly to ensure it fits their character concept.
- **Removal:** Powerful magic, divine intervention, or quest rewards might reverse mutations. Alternatively, embrace them as permanent character evolution.
- **Multiple Mutations:** Players can develop multiple mutations. Track each separately through its stages. A character might have Stage 4 Crystalline Eyes and Stage 2 Temporal Scarring simultaneously.
- **RP Opportunities:** Each stage offers roleplaying hooks. NPCs react to visible changes; casters might seek mentors or cures; mutations could become plot threads.
